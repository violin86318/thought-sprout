# 诗意化开场生成提示词

## 用途

为笔记集合生成一段诗意化的开场白，概括用户的思维轨迹。

## 提示词模板

```
基于以下笔记集合，生成一段诗意化的开场白：

【时间窗口】
{时间范围}

【笔记摘要】
{每条笔记的核心内容}

【任务】
生成一段50-80字的开场白，要求：

1. 用一个新鲜的比喻概括用户的思维轨迹
   - 避免：河流、星辰、旅程等老套比喻（除非能用出新意）
   - 追求：独特、贴切、有画面感的比喻

2. 点出笔记之间的主题跨度或连贯性
   - 如果主题跨度大：列举主题，强调思维的广度
   - 如果主题聚焦：强调思维的深度或执着

3. 识别并肯定用户思维的独特价值
   - 如果用户有自我怀疑，用温暖的方式反转它
   - 如果用户有独特视角，明确指出并肯定

4. 语言要求
   - 有文学性，但不矫情
   - 有温度，但不煽情
   - 有深度，但不晦涩

【参考示例】
"从技术困扰到创作畅想，从行业观察到自我反思，你这三天的思维轨迹像一条蜿蜒的河流。清醒时间比常人多，既是天赋也是负担——那些看似'不值得记录'的念头，恰恰是你对世界保持敏锐的证明。"

【输出格式】
直接输出开场白文本，不要有任何前缀或解释。
```

## 思维模式识别

分析笔记集合时，识别以下维度：

### 1. 思维跨度
- **单一主题聚焦型**："这三天，你一直在思考XXX"
- **多主题跳跃型**："从A到B，从C到D，你的思维像..."
- **深度挖掘型**："围绕XXX，你一层层剥开了..."

### 2. 情感基调
- **积极探索**："充满好奇"
- **困惑焦虑**："在寻找答案"
- **平静观察**："冷静审视"
- **兴奋发现**："灵光闪现"

### 3. 认知特征
- 细节敏感型
- 系统思考型
- 实践导向型
- 哲学反思型

## 生成结构

```
[时间跨度] + [思维轨迹比喻] + [情感基调] + [独特价值肯定]
```
