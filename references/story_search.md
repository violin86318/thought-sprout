# 故事素材检索策略

## 检索流程

对于每条笔记：

1. **提取核心关键词**（2-3个）

2. **构建检索查询**：
   - [关键词] + "历史" + "故事"
   - [关键词] + "科学家" + "发现"
   - [关键词] + "案例" + "转折"
   - [关键词] + "理论" + "名人"

3. **筛选标准**：
   - 有具体的人名、时间、地点
   - 有戏剧性的转折或对比
   - 有权威来源（书籍、研究、历史事件）

4. **故事改写**：
   - 保留核心事实
   - 增强戏剧性表达
   - 突出与用户笔记的关联

## 故事质量检查清单

### ✓ 具体性检查
- 有具体的人名？
- 有具体的时间（年份）？
- 有具体的事件或数据？

### ✓ 戏剧性检查
- 有冲突或转折？
- 有意外的结果？
- 有对比或反差？

### ✓ 相关性检查
- 与用户笔记主题强相关？
- 能够支撑"种子"中的观点？
- 不是生硬的知识堆砌？

### ✓ 准确性检查
- 历史事件是否准确？
- 人名、书名是否正确？
- 数据来源是否可靠？

## 三个视角的多维引用

当用户笔记中有金句或深刻感悟时，可以进行多视角扩展：

### 1. 科学视角
心理学、认知科学、社会学研究

**检索查询**：
- [核心概念] + "心理学研究"
- [核心概念] + "认知科学"

**示例**：
"刻意练习的任务难度要适中，能收到反馈，有足够的次数重复练习，学习者能够纠正自己的错误。" 
—— 安德斯·艾利克森，《刻意练习》作者

### 2. 历史视角
历史人物、商业案例、战略智慧

**检索查询**：
- [核心概念] + "历史人物"
- [核心概念] + "商业案例"
- [核心概念] + "战略"

**示例**：
"在战略方向没确定之前，是战略决定成败；在战略方向确定之后，是细节决定成败。" 
—— 王志纲，战略策划专家

### 3. 文化视角
哲学、艺术、匠心精神、文学

**检索查询**：
- [核心概念] + "哲学"
- [核心概念] + "匠心"
- [核心概念] + "文化"

**示例**：
"每一道工序，都是一次修行；每一个细节，都是对完美的致敬。" 
—— 匠心精神经典语录

## 筛选标准

- 有明确的引用来源
- 语言精炼，适合做金句
- 与用户观点形成呼应或对话
