# 单条笔记扩展提示词

## 用途

为单条笔记生成"种子→故事→Aha瞬间"的完整扩展。

## 提示词模板

```
为以下笔记生成"种子→故事→Aha瞬间"的完整扩展：

【用户笔记】
{笔记内容}

【检索到的素材】
{相关故事、案例、理论}

【任务】

1. 生成"种子"部分（80-120字）
   - 重新框定用户的观察，提供更宏大的视角
   - 制造认知冲突：看似是A，其实是B
   - 引出后续故事

   模板：
   [用户观察] + 看似是 [表面理解] + 但如果 [视角转换] + 这其实是 [更深层的本质]

2. 生成故事主体（200-300字）
   - 必须包含：具体的人名、时间、事件
   - 必须有：戏剧性的转折或对比
   - 必须有：与用户笔记的明确关联
   
   故事类型：
   - 历史转折故事：背景→危机→决策→结果
   - 科学发现故事：问题→科学家→实验→洞察
   - 文化现象故事：现象→背景→名人观点→当代意义

3. 生成"Aha瞬间"金句（20-40字）
   - 必须使用破折号、冒号或逗号制造停顿
   - 必须有反直觉的洞察
   - 必须与故事内容紧密相关
   
   模板：
   - 悖论式：[A]不是[B]，而是[C]
   - 反转式：[行为]不是[表面动机]，而是[深层逻辑]
   - 本质揭示式：[事物]的本质不是[A]，而是[B]
   - 讽刺式：[事物]最[形容词]的地方在于：[洞察]

【质量标准】
- 所有事实必须准确，不能编造
- 故事必须有戏剧性，不能平淡
- 金句必须有洞察，不能陈词滥调
- 语言必须有文学性，不能干巴巴

【输出格式】
🌱 种子
[种子内容]

[故事主体，分2-3段]

✨ Aha 瞬间
"[金句]"
```

## "种子"生成策略

### 1. 问题重定义
- 用户说：工程文件链接失效
- 种子说：这不是技术故障，而是"数字黑暗时代"的缩影

### 2. 视角提升
- 用户说：内存可能降价
- 种子说：这背后是一场"逆周期豪赌"

### 3. 本质揭示
- 用户说：想用AI模拟第二人格
- 种子说：这触及了AI领域最古老的问题——图灵测试

## 故事类型与结构

### 类型1：历史转折故事

**结构**：背景 → 危机 → 大胆决策 → 意外结果

**示例（三星内存战）**：
- 背景：1987年内存价格暴跌，所有厂商亏损
- 决策：三星用全年利润豪赌，继续扩产
- 结果：竞争对手倒闭，三星成为赢家
- 重复：2008年再次复制策略

**关键要素**：
- 具体数字（2.25美元跌到0.3美元）
- 具体公司名（奇梦达、尔必达）
- 戏剧化表达（"自杀式扩张"）

### 类型2：科学发现/理论

**结构**：问题 → 科学家 → 实验/理论 → 深层含义

**示例（图灵测试）**：
- 时间：1950年
- 人物：艾伦·图灵
- 实验：通过文字交流判断人机
- 深层含义：把哲学问题变成可操作实验

**关键要素**：
- 科学家的名字和年份
- 实验/理论的具体内容
- 反直觉的洞察

### 类型3：文化现象/名人观点

**结构**：现象 → 文化背景 → 名人观点 → 当代意义

**示例（严复翻译）**：
- 时间：1898年
- 人物：严复
- 理论："信达雅"三难
- 当代意义：台湾译本质量优势的根源

**关键要素**：
- 历史人物的具体贡献
- 经典理论或名言
- 与当代现象的连接

## 金句提炼模板

### 模板1：悖论式
```
[表面现象]不是[常规理解]，而是[深层本质]

示例：
"数字记忆比人类记忆更脆弱——它不会随时间淡去，而是会在某个瞬间彻底断裂。"
```

### 模板2：反转式
```
[行为]不是[表面动机]，而是[深层逻辑]

示例：
"在所有人都恐慌时加注，不是赌博，而是对周期规律的极致信仰。"
```

### 模板3：本质揭示式
```
[事物]的本质不是[A]，而是[B]

示例：
"翻译的本质不是文字转换，而是在两种文化之间搭建一座看不见的桥。"
```

### 模板4：讽刺/反思式
```
[事物]最[形容词]的地方在于：[反直觉的洞察]

示例：
"图灵测试最讽刺的地方在于：它测试的不是机器的智能，而是人类的判断力。"
```
